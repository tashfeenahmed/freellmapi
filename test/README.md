# Throttler Middleware Tests

This directory contains test files for the throttler middleware implementation. The tests verify that the throttler middleware is working correctly and provides the expected behavior.

## Test Files

- `test_throttler.ts`: Test script for the throttler middleware
- `test_results.md`: Test results for the throttler middleware
- `throttler_demo.md`: Demonstration of the throttler middleware behavior

## Running Tests

To run the tests, use the following command:

```bash
node test_throttler.js
```

This will execute the test script and output the results to the console.
